High Temperature Hydrogen Attack (HTHA)

This module calculates the High Temperature Hydrogen Attack (HTHA) Damage Factor ($D_{f}^{htha}$) based on material susceptibility, temperature, and hydrogen partial pressure.

Data Required for Determination of Susceptibility to HTHA

Required Data Comments Input Value
Required Data Material of construction Comments The component generic construction material (e.g. carbon steel, C-1/2 Mo, 2 1/4Cr-1 Mo). Input Value
Required Data Hydrogen partial pressure, MPa (psia) Comments Determine the hydrogen partial pressure, which is equal to the product of the mole fraction of hydrogen and the total pressure (absolute). Input Value
Required Data Temperature, °F (°C) Comments The temperature of exposure. Input Value
